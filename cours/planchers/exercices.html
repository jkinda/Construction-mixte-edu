Exercices Résolus

Applications pratiques de dimensionnement de planchers mixtes

Exercice 1
★ ★ ★

Plancher de Bureau - Vérification Complète

📋 Données
  • Portée sans étaiement : L = 3.2 m
  • Bac acier : Cofraplus 60 (0.88 mm)
  • Béton : C25/30
  • Épaisseur totale : ht = 140 mm (hc = 80 mm)
  • Charges supplémentaires : G' = 1.5 kN/m², Q = 2.5 kN/m²
  • Exigence feu : R60

Question : Vérifier le plancher en phase construction, puis en phase mixte (ELU), et dimensionner les armatures feu.

Étape 1 : Caractéristiques du bac

Cofraplus 60 (0.88 mm) :

  • \(A_p = 1284\) mm²/m
  • \(M_{pa,Rd} = 8.5\) kN.m/m
  • \(e_p = 30\) mm
  • Poids : 0.12 kN/m²
Étape 2 : Phase Construction

Charges :

\[g_{bac} = 0.12 \text{ kN/m²}\] \[g_{béton} = 25 \times 0.140 = 3.50 \text{ kN/m²}\] \[q_{chantier} = 1.5 \text{ kN/m²}\]

Combinaison ELU :

\[p_{Ed} = 1.35 \times (0.12 + 3.50) + 1.5 \times 1.5 = 7.14 \text{ kN/m²}\]

Moment sollicitant :

\[M_{Ed} = \frac{7.14 \times 3.2^2}{8} = 9.14 \text{ kN.m/m}\]

Vérification :

\[\frac{M_{Ed}}{M_{pa,Rd}} = \frac{9.14}{8.5} = 1.08 > 1.0 \quad \times\]

⚠️ Étaiement provisoire nécessaire (ou augmenter épaisseur du bac)

Étape 3 : Phase Mixte ELU

Avec étaiement, on vérifie la phase mixte.

Charges totales :

\[g_{tot} = 0.12 + 3.50 + 1.5 = 5.12 \text{ kN/m²}\]

Combinaison ELU :

\[p_{Ed} = 1.35 \times 5.12 + 1.5 \times 2.5 = 10.66 \text{ kN/m²}\]

Moment sollicitant :

\[M_{Ed} = \frac{10.66 \times 3.2^2}{8} = 13.6 \text{ kN.m/m}\]

Moment résistant plastique :

\[N_{pa} = 1284 \times 320 \times 10^{-3} = 410.9 \text{ kN/m}\] \[x_{pl} = \frac{410.9 \times 10^3}{0.85 \times 16.7 \times 1000} = 28.9 \text{ mm}\] \[z = 140 - 28.9/2 - 30 = 95.6 \text{ mm}\] \[M_{pl,Rd} = 410.9 \times 0.0956 = 39.3 \text{ kN.m/m}\]

Vérification :

\[\frac{M_{Ed}}{M_{pl,Rd}} = \frac{13.6}{39.3} = 0.35 < 1.0 \quad \checkmark\]
Étape 4 : Résistance au Feu R60

Vérification géométrique : hc = 80 mm ≥ 70 mm (exigence R60) ✓

Charge en situation de feu :

\[q_{fi} = 5.12 + 0.5 \times 2.5 = 6.37 \text{ kN/m²}\]

Moment :

\[M_{fi,Ed} = \frac{6.37 \times 3.2^2}{8} = 8.15 \text{ kN.m/m}\]

Armatures requises (ds ≈ 120 mm) :

\[A_s = \frac{8.15 \times 10^6}{0.9 \times 120 \times 500} = 151 \text{ mm²/m}\]

Choix : 1 HA 8 par nervure (5 nervures/m) → As = 251 mm²/m ✓

Exercice 2
★ ★ ★

Vérification Flèche et Vibrations

📋 Données
  • Portée : L = 4.0 m
  • Bac acier : Cofrastra 70 (0.88 mm)
  • Béton : C30/37
  • Épaisseur totale : ht = 160 mm
  • Charge permanente totale : g = 5.0 kN/m²
  • Charge variable : q = 3.0 kN/m²

Question : Vérifier la flèche ELS et la fréquence propre pour un usage bureaux.

Étape 1 : Coefficient d'équivalence
\[n_0 = \frac{E_a}{E_{cm}} = \frac{210\,000}{33\,000} = 6.36\]
Étape 2 : Inertie de la section mixte

Cofrastra 70 (0.88 mm) : Ip = 7.2 × 10⁵ mm⁴/m, hp = 73 mm

hc = 160 - 73 = 87 mm

Inertie béton :

\[I_c = \frac{1000 \times 87^3}{12} = 54.9 \times 10^6 \text{ mm}^4\text{/m}\]

Inertie équivalente (approximation) :

\[I_{eq} \approx I_p + \frac{I_c}{n_0} + A_p \cdot z^2\] \[I_{eq} \approx 7.2 \times 10^5 + \frac{54.9 \times 10^6}{6.36} + 1350 \times 80^2\] \[I_{eq} \approx 18.0 \times 10^6 \text{ mm}^4\text{/m}\]
Étape 3 : Calcul de la flèche

Charge ELS :

\[q_{ELS} = g + q = 5.0 + 3.0 = 8.0 \text{ kN/m²}\]

Flèche :

\[\delta = \frac{5 \times 8.0 \times 4000^4}{384 \times 210\,000 \times 18.0 \times 10^6}\] \[\delta = 7.0 \text{ mm}\]

Limite :

\[\delta_{lim} = \frac{L}{250} = \frac{4000}{250} = 16.0 \text{ mm}\]

\(\delta = 7.0 < 16.0\) mm ✓

Étape 4 : Fréquence propre

Masse linéique :

\[m = \frac{g}{9.81} \times 100 = \frac{5.0}{9.81} \times 100 = 51 \text{ kg/m}\]

Fréquence :

\[f_1 = \frac{\pi}{2 \times 4^2} \sqrt{\frac{210\,000 \times 18.0 \times 10^6}{51 \times 10^6}}\] \[f_1 \approx 7.6 \text{ Hz}\]

\(f_1 = 7.6 > 3\) Hz (bureaux) ✓

Exercice 3
★ ★ ★

Choix d'un Plancher pour Parking

📋 Données
  • Portée : L = 2.5 m (entre poutres)
  • Usage : Parking couvert
  • Charge variable : Q = 2.5 kN/m²
  • Exigence feu : R90

Question : Proposer un plancher adapté (bac + épaisseur) et vérifier la phase construction sans étaiement.

Étape 1 : Exigences R90

Pour R90, il faut hc ≥ 80 mm.

Choix initial : hc = 80 mm

Étape 2 : Choix du bac

Pour portée L = 2.5 m sans étaiement, un Cofraplus 60 (0.75 mm) devrait convenir.

  • hp = 60 mm
  • ht = 60 + 80 = 140 mm
  • Mpa,Rd = 6.7 kN.m/m
  • Poids = 0.10 kN/m²
Étape 3 : Vérification phase construction

Charges :

\[g_{béton} = 25 \times 0.140 = 3.50 \text{ kN/m²}\] \[p_{Ed} = 1.35 \times (0.10 + 3.50) + 1.5 \times 1.5 = 7.11 \text{ kN/m²}\]

Moment :

\[M_{Ed} = \frac{7.11 \times 2.5^2}{8} = 5.55 \text{ kN.m/m}\]

Vérification :

\[\frac{M_{Ed}}{M_{pa,Rd}} = \frac{5.55}{6.7} = 0.83 < 1.0 \quad \checkmark\]

✓ Pas d'étaiement nécessaire

Conclusion

Solution retenue :

  • Bac : Cofraplus 60 (0.75 mm)
  • Épaisseur totale : 140 mm
  • Béton : C25/30
  • Armatures feu : 1 HA 10/nervure pour R90
