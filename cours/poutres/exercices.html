Exercice 1
★ ★ ★

Poutre de Plancher de Bureau

Objectif : Dimensionner une poutre mixte de plancher de bureau en vérifiant l'ELU et l'ELS.

📋 Données
  • Portée : \(L = 10\) m (travée simple)
  • Entraxe entre poutres : 3.0 m
  • Dalle béton C30/37 : épaisseur \(h_c = 120\) mm
  • Bac acier Cofraplus 60 : \(h_p = 60\) mm
  • Charges permanentes (hors PP) : \(G' = 1.5\) kN/m²
  • Charges d'exploitation : \(Q = 3.5\) kN/m² (bureaux)
  • Profilé envisagé : IPE 400, acier S355

Questions :

  1. Calculer les charges et le moment sollicitant ELU
  2. Déterminer la largeur participante
  3. Vérifier le moment résistant plastique
  4. Calculer le nombre de goujons requis
  5. Vérifier la flèche ELS
Étape 1 : Évaluation des Charges

Poids propre de la dalle mixte :

\[g_{dalle} = (h_c + h_p/2) \times 25 = (0.12 + 0.03) \times 25 = 3.75 \text{ kN/m²}\]

Charges linéiques sur la poutre :

\[g_k = (3.75 + 1.5) \times 3.0 = 15.75 \text{ kN/m}\] \[q_k = 3.5 \times 3.0 = 10.5 \text{ kN/m}\]

Estimation poids propre poutre IPE 400 : ≈ 1.0 kN/m

Charge pondérée ELU :

\[p_{Ed} = 1.35 \times (15.75 + 1.0) + 1.5 \times 10.5 = 22.61 + 15.75 = 38.36 \text{ kN/m}\]

Moment sollicitant :

\[\boxed{M_{Ed} = \frac{p_{Ed} \cdot L^2}{8} = \frac{38.36 \times 10^2}{8} = 479.5 \text{ kN.m}}\]
Étape 2 : Largeur Participante

Pour une travée simple : \(L_e = L = 10\) m

\[b_{ei} = \min\left(\frac{10000}{8}, \frac{3000}{2}\right) = \min(1250, 1500) = 1250 \text{ mm}\] \[b_{eff} = 2 \times 1250 = 2500 \text{ mm}\]
Étape 3 : Moment Résistant Plastique

Données IPE 400, S355 : \(A_a = 84.5\) cm², \(h_a = 400\) mm

Forces plastiques :

\[N_{a,pl} = 84.5 \times 10^{-4} \times 355 \times 10^3 = 3000 \text{ kN}\] \[N_{c,f} = 0.85 \times 20 \times 2500 \times 120 \times 10^{-3} = 5100 \text{ kN}\]

\(N_{c,f} > N_{a,pl}\) → ANP dans la dalle

Position ANP :

\[z_{pl} = \frac{3000 \times 10^3}{0.85 \times 20 \times 2500} = 70.6 \text{ mm} < 120 \text{ mm ✓}\]

Bras de levier :

\[d = \frac{400}{2} + 120 + 60 - \frac{70.6}{2} = 344.7 \text{ mm}\]

Moment résistant :

\[\boxed{M_{pl,Rd} = 3000 \times 0.3447 = 1034 \text{ kN.m}}\]

Vérification :

\[\frac{M_{Ed}}{M_{pl,Rd}} = \frac{479.5}{1034} = 0.46 < 1.0 \quad \checkmark\]
Étape 4 : Connexion

Effort à transférer : \(V_\ell = N_{a,pl} = 3000\) kN

Résistance goujons ∅19, \(f_u = 450\) MPa :

\[P_{Rd} = 81.7 \text{ kN}\]

Nombre par demi-travée :

\[n = \frac{3000}{81.7} = 36.7 \rightarrow \boxed{37 \text{ goujons}}\]

Soit 74 goujons sur toute la poutre, entraxe ≈ 135 mm

Étape 5 : Vérification Flèche ELS

Coefficient d'équivalence : \(n_0 = 6.36\)

Inertie mixte : \(I_{eq} \approx 92\,240\) cm⁴

Flèche :

\[\delta = \frac{5 \times 26.25 \times 10^4}{384 \times 210 \times 92240} = 17.6 \text{ mm}\]

Limite :

\[\delta_{lim} = \frac{L}{250} = 40 \text{ mm}\] \[\boxed{\delta = 17.6 \text{ mm} < 40 \text{ mm} \quad \checkmark}\]
Exercice 2
★ ★ ★

Calcul de Goujons

Objectif : Calculer la résistance d'un goujon et déterminer le nombre requis.

📋 Données
  • Goujon : ∅22 mm, hauteur 125 mm après soudage
  • Résistance traction : \(f_u = 450\) MPa
  • Béton C35/45 : \(f_{ck} = 35\) MPa, \(E_{cm} = 34000\) MPa
  • Effort à transférer : \(V_\ell = 3500\) kN (par demi-travée)
Calcul de la résistance du goujon

Coefficient α :

\[\frac{h_{sc}}{d} = \frac{125}{22} = 5.68 > 4 \Rightarrow \alpha = 1.0\]

Rupture acier :

\[P_{Rd,1} = \frac{0.8 \times 450 \times \pi \times 22^2/4}{1.25 \times 1000} = 109.5 \text{ kN}\]

Écrasement béton :

\[P_{Rd,2} = \frac{0.29 \times 1.0 \times 22^2 \times \sqrt{35 \times 34000}}{1.25 \times 1000} = 121.8 \text{ kN}\] \[\boxed{P_{Rd} = \min(109.5, 121.8) = 109.5 \text{ kN}}\]

Nombre de goujons :

\[n = \frac{3500}{109.5} = 32.0 \rightarrow \boxed{32 \text{ goujons par demi-travée}}\]
Exercice 3
★ ★ ★

Poutre avec Connexion Partielle

Objectif : Vérifier une poutre mixte en connexion partielle.

📋 Données
  • IPE 330, S355, portée 8 m
  • Dalle C30/37, \(h_c = 100\) mm, \(b_{eff} = 2000\) mm
  • 20 goujons ∅19 par demi-travée (\(P_{Rd} = 81.7\) kN)
  • Moment sollicitant : \(M_{Ed} = 350\) kN.m
Étape 1 : Forces plastiques

IPE 330 : \(A_a = 62.6\) cm², \(W_{pl,y} = 804\) cm³

\[N_{a,pl} = 62.6 \times 10^{-4} \times 355 \times 10^3 = 2222 \text{ kN}\] \[N_{c,f} = 0.85 \times 20 \times 2000 \times 100 \times 10^{-3} = 3400 \text{ kN}\]
Étape 2 : Degré de connexion

\(V_\ell = N_{a,pl} = 2222\) kN (connexion complète)

\[\eta = \frac{n \times P_{Rd}}{V_\ell} = \frac{20 \times 81.7}{2222} = 0.735\]

Vérification du degré minimal :

\[\eta_{min} = 1 - \frac{355}{355}(0.75 - 0.03 \times 8) = 1 - 0.51 = 0.49\] \[\eta = 0.735 > 0.49 \quad \checkmark\]
Étape 3 : Moment résistant réduit

Moment acier seul :

\[M_{a,pl,Rd} = W_{pl,y} \times f_{yd} = 804 \times 10^{-3} \times 355 = 285.4 \text{ kN.m}\]

Moment connexion complète (calculé à 580 kN.m)

Moment réduit :

\[M_{Rd} = 285.4 + 0.735 \times (580 - 285.4) = 285.4 + 216.5 = 502 \text{ kN.m}\]

Vérification :

\[\frac{M_{Ed}}{M_{Rd}} = \frac{350}{502} = 0.70 < 1.0 \quad \boxed{\checkmark}\]
